List of HTML Tags:

  1. <html>: The root element that encapsulates the entire HTML document.
  2. <head>: Contains metadata elements such as title, character set, and links to external resources.
  3. <title>: Specifies the title of the HTML document, displayed in the browser's title bar or tab.
  4. <meta>: Provides metadata about the document, including character encoding and author information.
  5. <link>: Defines relationships between the current document and external resources like stylesheets.
  6. <style>: Contains CSS rules to style the document's elements.
  7. <script>: Embeds JavaScript code within the HTML document for interactivity.
  8. <body>: Encloses the visible content of the web page, including text, images, and links.
  9. <h1> - <h6>: Headings to structure content and indicate hierarchy.
  10. <p>: Defines a paragraph of text to separate content into readable blocks.
  11. <a>: Creates hyperlinks to other web pages or resources for navigation and interactivity.
  12. <img>: Embeds images in the document to enhance visual content.
  13. <ul>: Creates an unordered (bulleted) list to group related items without a specific order.
  14. <ol>: Creates an ordered (numbered) list to organize items with a specific order.
  15. <li>: Represents a list item within <ul> or <ol> elements.
  16. <div>: A generic container for grouping and styling elements for structure and formatting.
  17. <span>: A generic inline container for styling specific portions of text for targeted styling within larger blocks of text.
